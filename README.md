# LoadObjFiles-Threejs-Vue2（local data）

A basic example to load and show 3D object files (.obj) to scene



## Requirements

```
> Node 12.22

> Vue 2
```

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```
